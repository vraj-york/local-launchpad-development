🎯 Feedback Widget Test Page

Welcome to the Feedback Widget Demo!

This page demonstrates the feedback widget functionality. Look for the purple button in the bottom-right corner.

Keyboard Shortcut: Press Ctrl+Shift+F (or Cmd+Shift+F on Mac) to open the feedback widget.

📸 Screenshot Capture

  • Capture visible area
  • Capture full page
  • Automatic widget hiding
  • High-quality PNG output

✏️ Annotation Tools

  • Pen/Drawing tool
  • Arrow annotations
  • Rectangle shapes
  • Text labels
  • Color picker (6 colors)
  • Undo/Redo support

📝 Feedback Form

  • Description field (required)
  • Character counter
  • Auto-collected metadata
  • Browser information
  • Screen resolution
  • Page URL tracking

How to Use

Step 1: Click the purple feedback button in the bottom-right corner (or press Ctrl+Shift+F)

Step 2: Choose to capture the visible area or full page

Step 3: Annotate the screenshot with drawing tools (or skip)

Step 4: Add a description of your feedback

Step 5: Submit and receive a Jira ticket number

Integration Code

To add this widget to your own page, include these scripts:

<script src="https://cdn.yourplatform.com/feedback-widget.min.js"></script> <script> FeedbackWidget.init({ projectId: 'your-project-id', apiUrl: 'https://api.yourplatform.com', onSuccess: (data) => { console.log('Feedback submitted:', data); }, onError: (error) => { console.error('Submission failed:', error); } }); </script>
